# Intern Feedback Sentiment Analysis

This project uses Natural Language Processing (NLP) and Machine Learning to analyze intern feedback and classify sentiments as Positive or Negative.

## Technologies Used

* Python
* Scikit-learn
* TF-IDF Vectorization
* Logistic Regression

## Features

* Sentiment Classification
* Feedback Analysis
* Model Saving using Joblib

## Dataset

Synthetic intern feedback dataset created for educational purposes.

## Future Improvements

* Add Neutral Sentiment
* Use Transformer Models (BERT)
* Deploy as Web Application
