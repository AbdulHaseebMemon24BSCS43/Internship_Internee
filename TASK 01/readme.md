# Task 01 - Intern Performance Prediction

## Objective

Build a Machine Learning model to predict intern performance based on task completion time, feedback ratings, and attendance records.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Random Forest Regressor

## Dataset Features

* Completion_Time
* Feedback_Rating
* Attendance
* Performance_Score

## Features

* Data Preprocessing
* Missing Value Handling
* Model Training & Prediction
* Performance Evaluation
* Model Saving using Joblib

## Workflow

Dataset → Data Preprocessing → Train/Test Split → Random Forest Model → Performance Prediction
